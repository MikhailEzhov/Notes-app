## Run the Notes-app locally

### Step 1: Clone project

```sh
git clone https://github.com/MikhailEzhov/Notes-app.git Notes-app
cd Notes-app
```

### Step 2: Install npm dependencies

```sh
npm install
```

### Step 3: Usage

```sh
npm start
```
